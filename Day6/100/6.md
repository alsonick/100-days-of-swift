## Day 6 - Topics Summary

- Creating basic closures ✅

  - Functions with no name are called closures.
  - Results: 12/12

- Accepting parameters in a closure ✅

  - Closures in Swift can also accept parameters.
  - Closures that accept parameters cannot use external parameter labels.
  - Results: 11/12

- Returning values from a closure ✅

  - You can return values from closures in Swift.
  - To return the value, we need to use `-> String` before the `in` keyword.
  - Results: 11/12

- Closures as parameters ✅

  - You can pass closures into functions in Swift.
  - Results: 12/12

- Trailing closure syntax ✅

  - If the last parameter to a function is a closure, Swift lets you use special syntax called _trailing closure syntax_.
  - Rather than pass in your closure as a parameter, you pass it directly after the function inside braces.
  - Results: 11/12
