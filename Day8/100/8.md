## Day 8 - Topics Summary

- Creating your own structs ✅

  - You can design on your own types using structures, or more commonly known as just _structs_
  - You use the `struct` keyword to define a struct.
  - Results: 11/12

- Computed properties ✅

  - Computed properties in Swift are properties that can run code and return values.
  - Results: 10/12

- Property observers ✅

  - Property observers let you run code before or after any property changes.
  - Results: 10/12

- Methods ✅

  - Functions inside structs are called methods.
  - Results: 12/12

- Mutating methods ✅

  - If a struct has a variable property but the instance of the struct was created as a constant, that property cannot be changed.
  - When you need to change a property inside a method, you need to mark it using the `mutating` keyword.
  - Results: 11/12

- Properties and methods of strings ✅

  - Strings are structs.
  - Results: 5/6

- Properties and methods of arrays ✅

  - Arrays are also structs.
  - Results: 11/12
