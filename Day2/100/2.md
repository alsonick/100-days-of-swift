## Day 2 - Topics Summary

- Arrays ✅

  - Learned about Arrays in Swift.
  - To access an element in an array you must use the square bracket syntax.
  - Results: 6/6

- Sets ✅

  - To create a set you must pass in an array.
  - Sets can't have duplicates.
  - You can't access elements inside a set with the square bracket syntax.
  - Results: 11/12

- Tuples ✅

  - You can't add or remove items in a tuple.
  - Tuples always have a fixed size.
  - You can access elements in a tuple using numerical positions or by using their name.
  - Once a tuple is created, you can change the values but not the types.
  - Results: 6/6

- Arrays vs sets vs tuples ✅

  - If you need a fixed collection of related values then you should use a tuple.
  - If you need a collection of unique values or if you need to a find a specific item quickly then you should use a set.
  - If you need a collection that _can_ contain duplicates then you should use an array.
  - Arrays are the most commonly used.
  - Results: 4/6

- Dictionaries ✅

  - Dictionaries are also collections like arrays.
  - Each item in a dictionary is separated by a comma.
  - Dictionaries are written in brackets with a colon between your identifier and value.
  - Results: 6/6

- Dictionary default values ✅

  - If you try to access a value from a dictionary that doesn't exist, Swift will return nil.
  - You can add a default value whenever try to access a value from a dictionary.
  - Results: 10/12

- Creating empty collections ✅

  - To create an empty collection just write its type followed by opening and closing parentheses.
  - The exception is creating an empty set, which is done like this: `var words = Set<String>()`.
  - Results: 6/6

- Enumerations ✅

  - Enums are a way of defining groups of related values together.
  - Results: 6/6

- Enum associated values ✅

  - Enums can also store _associated_ values attached to each case.
  - Results: 6/6

- Enum raw values ✅

  - Swift automatically assigns each case with a number starting with 0.
  - You can assign a case a specific value.
  - Results: 6/6

- Complex types: Summary ✅
  - Results: 5/6
