## Day 9 - Topics Summary

- Initializers ✅

  - Initializers are special methods that provide different ways to crate your struct.
  - All structs come with one by default, called their memberwise initializer.
  - Results: 11/12

- Referring to the current instance ✅

  - You can refer to the current instance of a struct using the `self` keyword.
  - Results: 11/12

- Lazy properties ✅

  - Swift lets you create some properties only when they are needed.
  - Lazy properties are created using the `lazy` keyword.
  - Results: 6/6

- Static properties and methods ✅

  - You can access a property or struct without initializing it by making them as `static`.
  - Results: 11/12

- Access control ✅

  - Access control lets you restrict which code can use properties and methods.
  - Results: 10/12
